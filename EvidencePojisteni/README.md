# Evidence pojištěných

Konzolová aplikace vytvořená v jazyce C#.

## Funkce

- Přidání pojištěného
- Výpis všech pojištěných
- Vyhledání podle jména a příjmení

## Použité technologie

- C#
- .NET 10
- Objektově orientované programování

## Struktura projektu

- Program.cs - spuštění aplikace
- Menu.cs - uživatelské menu
- Evidence.cs - práce s daty
- Pojisteny.cs - datový model
- ConsoleHelper.cs - pomocné metody
